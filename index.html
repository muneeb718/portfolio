AI Project

Architecture

Three-layer pipeline: (1) ML layer — TF-IDF vectorizer + Logistic Regression for linguistic scoring, (2) Rule-based heuristics for clickbait & sensationalism detection, (3) Domain reputation scoring against trusted source database.

Technologies

Python 3.x Flask scikit-learn TF-IDF Vectorizer Logistic Regression NLP Pipeline Pandas / NumPy

Key Features

  • Sub-200ms response time via optimized Flask API
  • Forensic dashboard with radar charts & phrase-level highlighting

  • Hard safety overrides for known misinformation domains
  • Domain whitelist/blacklist reputation scoring system
  • Real-time analysis with detailed confidence breakdown

Gesture Control Car
Hardware

Problem Statement

Traditional RC car controllers require physical joysticks. A gesture interface creates a more intuitive, hands-free experience using natural hand movements.

Architecture

Transmitter ESP32 reads 3-axis tilt from MPU6050, maps gestures to directional commands, and broadcasts via ESP-NOW (connectionless WiFi). Receiver ESP32 maps commands to PWM signals driving the L298N for motor direction and speed control.

Technologies

ESP32 DevKit (×2) MPU6050 IMU Sensor L298N Dual H-Bridge Arduino IDE ESP-NOW Protocol Li-ion Battery Pack 4× DC Gear Motors

Key Features

  • Sub-50ms latency from gesture to motion
  • ESP-NOW protocol for low-overhead P2P communication
  • 4-directional control mapped to tilt angles
  • Speed modulation via tilt magnitude
  • No router required — direct device-to-device WiFi
  • Stable dual-battery power management system

Hotel Network Infrastructure
Network Design

Problem Statement

Hotels need secure, segmented networks separating guest WiFi, management systems, and operational infrastructure across multiple floors.

Architecture

3-tier hierarchy: Core (main router + ISP link), Distribution (L3 switch with inter-VLAN routing + DHCP), Access (floor switches). VTP for VLAN propagation, STP for loop prevention.

Technologies

Cisco Packet Tracer Cisco Catalyst Switches Cisco ISR Router Layer 3 Switch Wireless APs DHCP Server VTP Protocol

Key Features

  • 5 isolated VLANs: Guest, Staff, Management, CCTV, POS
  • Inter-VLAN routing via Layer 3 switch (no router-on-stick)
  • Automated IP assignment with centralized DHCP pools
  • Wireless coverage per floor via managed APs
  • Internet connectivity with NAT overload (PAT)
  • VLAN-based security segmentation for PCI compliance
